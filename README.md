# Athena Plugin - Pizza Delivery Job

A pizza delivery job plugin for the Athena Framework compatible with `4.0.0` of the [Athena Framework](https://athenaframework.com/).

## Installation

1. Open a command prompt in your main Athena Directory.
2. Navigate to the plugins folder.

```ts
cd src/core/plugins
```

3. Copy the command below.

**SSH**

```
git clone git@github.com:Stuyk/athena-plugin-job-pizza-delivery.git
```

**HTTPS**
```
git clone https://github.com/Stuyk/athena-plugin-job-pizza-delivery
```

4. Start the Server
